Tradutor que utiliza Vue.js e Yandex Translate API, 
Inspirado em um projeto do Brad Traversy.

# vue-translate

## Setup do Projeto
```
npm install
```

### Compila e faz o hot-reload para desenvolvimento
```
npm run serve
```

### Compila e minifica para produção
```
npm run build
```

### Executa os teste
```
npm run test
```

### Corrige arquivos
```
npm run lint
```
